What is Promptify?

In the world of AI, "garbage in equals garbage out" - which is why refined, well-crafted prompts are essential for getting intelligent, accurate responses.

Promptify is a lightning-fast Chrome extension that revolutionizes how you interact with AI applications. Transform basic prompts into sophisticated, detailed instructions that unlock the full potential of AI models.

Instant Activation

Simply highlight any text and click the popup button to refine your prompt instantly - no waiting, no complexity.

Smart Refinement

Control the level of detail with 1-5 refinement iterations - choose between lightning speed or maximum precision.

Zero Hassle

No sign-ups, no subscriptions, no accounts. Just pure speed and efficiency with your own API key.

Step-by-Step Setup Guide

Follow these steps to get Promptify up and running in just a few minutes. Each step is designed to be simple and straightforward.

Important Note

Make sure to save your settings immediately after entering your API key. Settings will not function properly if no API key has been entered before pressing Save Settings.

Download From Chrome Store

Install Promptify from the Chrome Web Store to get started. The extension is lightweight and installs instantly.

Download from Chrome Store

Go to ChatGPT.com or Claude.ai

Navigate to your preferred AI platform. Promptify works seamlessly with ChatGPT, Claude, and other major AI applications.

Press Ctrl + B to Access Settings

Use the default keyboard shortcut Ctrl + B to open Promptify settings. You can customize this shortcut later!

Click Google AI Studio Link

In the settings modal, find the "Gemini API Key" section and click the Google AI Studio button to get your API key.

Follow API Key Generation Steps

On Google AI Studio, generate your API key. If you need detailed guidance, visit: merge.dev API key guide

Copy & Paste API Key

Copy your generated API key from Google AI Studio and paste it into the "Gemini API Key" field in Promptify settings.

IMPORTANT: Save Settings Immediately

Critical Step: Click "Save Settings" right after entering your API key. The extension will not function properly if settings are not saved with a valid API key.

Customize Promptify

Reopen settings (Ctrl + B) to customize refinement iterations, keyboard shortcuts, and select your preferred enhancement engine.

Highlight & Refine!

Highlight any text you want to improve, click the popup button, and watch your basic prompt transform into a sophisticated, detailed instruction!

Refinement Iterations Explained

1-2

Speed Focus

Quick refinements for basic prompt improvements. Perfect for simple tasks and when you need lightning-fast results.

3

Balanced

The sweet spot between speed and quality. Recommended for most everyday use cases and general improvements.

4-5

Maximum Precision

Highly detailed, sophisticated prompts. Best for complex tasks requiring nuanced, professional-grade instructions.
